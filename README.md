# Lizmo
A little helper app for my mom — one installable home screen with simple tools that make her day easier.

**Live:** https://yinzermcmullet.github.io/lizmo/ (hosted on GitHub Pages)

- `index.html` — the home screen / tool menu (add or retire a tool by editing the `TOOLS` list).
- `tools/<name>/` — each tool is its own self-contained folder.
